GitHub Profile Viewer
A high-performance React application that interacts with the GitHub REST API to fetch and display real-time user data. 
This project was developed as part of a Professional Frontend Internship Portfolio.
Random Profile Discovery Uses logic to fetch random users to improve platform engagement.
Fast Bundling powered by parcel for lightning-fast build times and hot reloading.
Tech Stack
Library React.js (v19)
Bundler Parcel.
Styling Modular CSS3.
API GitHub REST API.
